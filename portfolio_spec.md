
# PORTFOLIO PROJECT — FULL SPECIFICATION (for Cursor)
Author: Awad Issa

## 1. Project Overview
Create a high-end professional portfolio website for Awad Issa with a modern, premium, dark-tech aesthetic using smooth animations and clean layout.

## 2. Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- ShadCN UI
- Framer Motion
- Lucide Icons
- Fonts: Inter, Cairo
- Deployment: Vercel

## 3. Design Principles
### Colors
- Background: #0d0d0d
- Text: #f2f2f2
- Muted: #b3b3b3
- Accent: Indigo or Cyan

### Layout
- Max width: 1200px
- Large spacing
- Rounded corners (xl/2xl)
- Soft shadows

### Typography
- Large headings, clean spacing

### Animations
- Framer Motion fade-in, subtle movement, hover effects
